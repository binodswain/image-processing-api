Image Processing API

This is a simple image processing API that allows you to resize images.

Frontend endpoints

  • /upload

    UI to upload image to full image assets folder

  • /fullimages

    UI to view full size image available

  • /thumbnails

    UI to view processed thumbnail images

API endpoints

  • /api/images

    To get resized image from request query params.
    /api/images?filename=encenadaport&height=100&width=100

    To get list of fullsize images available. /api/images

    To get single fullsize image. /api/images?filename=encenadaport

  • /api/upload

    [POST] To upload original fill size image from UI

  • /api/thumbnails

    To get list of processed thumbnail image and url path
